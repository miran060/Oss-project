## 문제1
```
public class plusMinus {
	public static void main(String[] args) {
		int Var1 = 4;
		int Var2 = 3;
		int Var3 = 2 * Var1++ - --Var2;
		int Var4 = Var2 * 2; 

		int result = Var1 + Var3++ * ++Var2 / Var4;
		
		System.out.println(result);
	}
}
```
**정답 : 9**
