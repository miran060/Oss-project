### 1. 다음 코드를 보고 출력되는 값을 구하시오.

```
public class IncrementTest{
    public static void main(String[] args){

        int num1 = 7;
        int num2 = 16;
        int num3 = 38;

        int result1 = num1++ + num2-- + num1-- + ++num2;
        int result2 = --num1 + num3-- - num3++ + num2;

        System.out.println("result1 = " + result1);
        System.out.println("result2 = " + result2);
    }
}
```
<details>
  <summary><b>정답 확인</b></summary>
  
  result1 = 47 <br>
  result2 = 23
</details>



<details>
  <summary><b>해설</b></summary>
    
  전위 연산자: 연산자 ++가 피연산자 n보다 앞에 위치할 때, 1증가 후 연산 <br>
  후위 연산자: 연산자 ++가 피연산자 n보다 뒤에 위치할 때, 연산 후 1증가


### result1

```
1) num1++ : 7 사용  → num1 = 8
2) num2-- : 16 사용 → num2 = 15
3) num1-- : 8 사용  → num1 = 7
4) ++num2 : num2 = 15 → 전위 증가(+1) 후 16 사용 → num2 = 16
```

**result1 = 7 + 16 + 8 + 16 = 47**

---

### result2

```
1) --num1 : num1 = 7 → 전위 감소(-1) 후 6 사용 → num1 = 6
2) num3-- : 38 사용 → num3 = 37
3) num3++ : 37 사용 → num3 = 38
4) num2   : 16 사용
```

**result2 = 6 + 38 - 37 + 16 = 23**

</details>

### 2. 다음 코드를 보고 출력되는 값을 구하시오
```
public class IncrementTest2 {
    public static void main(String[] args) {

        int a = 5;
        int b = 12;
        int c = 3;

        int result1 = a++ + --b * c++ - a;
        int result2 = ++a + (b-- + ++c) - b++ * a--;

        System.out.println("r1 = " + r1);
        System.out.println("r2 = " + r2);
    }
}

```
<details>
  <summary><b>정답 확인</b></summary>
  
  result1 = 47 <br>
  result2 = 23
</details>

<details>
  <summary><b>해설</b></summary>
  전위 연산자: 연산자 ++가 피연산자 n보다 앞에 위치할 때, 1증가 후 연산 <br>
  후위 연산자: 연산자 ++가 피연산자 n보다 뒤에 위치할 때, 연산 후 1증가
  
### result1
```
1) a++  : 5 사용 → a = 6
2) --b  : b = 11 → 11 사용
3) c++  : 3 사용 → c = 4
4) --b * c++ = 11 * 3 = 33
5) - a  : 현재 a = 6 → -6
```
**result1 = 7 + 16 + 8 + 16 = 47**

---

### result2
```
1) ++a  : a = 7 → 7 사용
2) b--  : 11 사용 → b = 10
3) ++c  : c = 5 → 5 사용
4) (b-- + ++c) = 11 + 5 = 16
5) b++ * a-- = 10 * 7 = 70  → 이후 b = 11, a = 6
```
**result2 = 6 + 38 - 37 + 16 = 23**

</details>
