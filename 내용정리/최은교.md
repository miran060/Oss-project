# 자바 1~6강 정리

## 1. 변수
- 첫 글자는 문자로 중간에 $, _ 포함 가능
- 변수 명은 카멜케이스로 이루어진 영문자

### 변수 초기화
- int iValue;
- 지역 변수는 반드시 초기화 필요
- 전역변수는 자동으로 기본값 초기화

### 변수 타입
| 변수 구분 | 기본 타입 | 메모리 크기 |
|----------|-----------|-------------|
|정수 |byte|1byte|
|    |char, short|2byte|
|    |int|4 byte|
|    |long|8byte|
|실수|float|4 byte|
||double|8 byte|
|논리 값|boolean|1 byte|
|문자열|String||
