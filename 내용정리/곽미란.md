3. 변수와 타입
✔ 변수 선언 규칙

타입 + 변수명

첫 글자는 문자 또는 _ 또는 $만 가능

숫자로 시작 불가

camelCase 사용 권장

예약어( int, class 등 ) 사용 불가

✔ 기본 타입(Primitive Type)
🔹 정수 타입
타입	메모리	범위
byte	1byte	-128 ~ 127
short	2byte	-32,768 ~ 32,767
char	2byte	0 ~ 65535 (유니코드)
int	4byte	약 ±21억
long	8byte	매우 큼 (접미사 L 사용)
🔹 문자 타입
char c1 = 'A';  
char c2 = 65;   // 'A'
char c3 = ' ';  // 공백 가능


⚠ 큰따옴표 사용 시 오류

char c = "A"; // 오류

🔹 실수 타입
타입	메모리	정확도
float	4byte	7자리 정밀도
double	8byte	15자리 정밀도
🔹 논리 타입
boolean flag = true;

🔹 문자열(String)
String name = "홍길동";

✔ 이스케이프 문자
문자	의미
\t	탭
\n	줄바꿈
"	큰따옴표 출력
\	역슬래시 출력
✔ 타입 변환
🔹 자동 타입 변환

작은 크기 → 큰 크기
byte < short, char < int < long < float < double

🔹 강제 타입 변환(캐스팅)
int x = (int) 3.14; // 3

🔹 문자열 → 기본 타입
int n = Integer.parseInt("10");
double d = Double.parseDouble("3.14");
boolean b = Boolean.parseBoolean("true");

✔ 출력
System.out.println("줄바꿈 O");
System.out.print("줄바꿈 X");
System.out.printf("이름: %s, 나이: %d", name, age);

8. 클래스(Class)와 객체(Object)
✔ 클래스 정의
class Person {
    String name;
    int age;

    void introduce() {
        System.out.println(name + ", " + age);
    }
}

✔ 객체 생성
Person p = new Person();
p.name = "홍길동";
p.age = 20;
p.introduce();

✔ 생성자(Constructor)
class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}


사용:

Person p = new Person("김철수", 25);

✔ 메소드(Method)

    return x + y;
}

✔ this 키워드
