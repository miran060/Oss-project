# Oss-project
오픈소스소프트웨어 팀과제

##### 주제 :
자바 기본 코드 및 내용 정리  
##### 주제 선정 이유 :
파이썬,C언어에서 배운 것과는 다른 코드의 형태로 코드 작성에 대한 어려움을 겪음. 수업 개념 및 어려웠던 코드 등을 작성하고 공유함으로써 학업에 서로서로 도움을 줄 수 있을것으로 기대됨  
<br>
<br>
자바 수업 때 사용하는 '이것이 자바다' 책, 하만석 교수님이 제공해주시는 피피티를 활용함  
1장~6장까지의 내용을 정리할 것임  
<br>
<br>
자바의 모든 내용을 담기보다는 기초적인 내용과 코드 몇가지로 여러 깃허브의 기능을 활용해 볼 예정  
이슈, 마일스톤, 레이블, PR, 위키, 프로젝트 등
